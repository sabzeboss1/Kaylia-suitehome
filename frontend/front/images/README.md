# Images Directory

Ce répertoire contient toutes les images du site Kaylia Suite Home.

## Structure

- **apartments/** - Images des appartements et studios
- **spa/** - Images du spa et des services
- **hero/** - Images pour les sections hero/bannières
- **gallery/** - Images de la galerie
- **icons/** - Icônes et logos

## Format recommandé

- Format: JPG pour les photos, PNG pour les logos/icônes
- Optimisation: Compresser les images avant upload
- Nommage: Utiliser des noms descriptifs en minuscules avec tirets (ex: `studio-chambre-1.jpg`)

## Tailles recommandées

- Hero images: 1920x1080px
- Apartment images: 1200x800px
- Thumbnails: 400x300px
- Icons: 64x64px ou SVG
